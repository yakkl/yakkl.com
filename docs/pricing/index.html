Yakkl vs. Slack vs. Microsoft Teams?

Slack is a very good tool. In fact, I brought Slack Enterprise into a large public enterprise but ran into “channel sprawl” very fast. Channel Sprawl is how I describe the fact that you need to create a channel in Slack for everything. Meaning, if I have a team that may be working on a few different projects along with other teams then I will have to create a channel for ‘Team 1 - Project 1’ and then another channel for ‘Team 1 - Project 2’ and so on. There are many best practices according to Slack on how to handle this but you still have “Channel Sprawl”.
