What versions of Hugo do you support?

Yakkl supports all stable releases of Hugo. If your project requires a version that is not available in the version dropdown, please let us know and we can add it.

Extensions

Asset Pipeline

Yakkl supports all Hugo’s pipes including Sass compilation, postCSS, minification, bundling and fingerprinting.

Content Formats

We also support the following software to enable Hugo’s optional content formats:

  • asciidoc
  • asciidoctor
  • pandoc
