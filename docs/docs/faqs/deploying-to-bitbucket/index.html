Why Can't I Deploy to a Bitbucket Branch?

Bitbucket allows you to host a website by pushing static HTML to a branch in your Bitbucket repo, much like GitHub Pages. Previously, users with a Bitbucket repo could configure their site to be deployed back to Bitbucket, much like our GitHub Pages deployment option.

This behavior relied on using short-lived OAuth tokens to authenticate your account with Bitbucket, which caused problems when deployments were triggered while you were not logged in to Yakkl. We have since upgraded our platform to use long-lived SSH keys instead of OAuth tokens.

Unfortunately, due to limitations of Bitbucket’s API, we are no longer able able to support deployments to Bitbucket at this time. Due do these limitations, we recommend using GitHub or GitLab to provide the optimum experience with Yakkl.
