2018/09/12 Changelog

Enhancements

  • IPV6: Yakkl now fully support for IPV6
  • Jekyll Previews: Improved support for users previewing Jekyll sites using Gem-based themes.

Bug Fixes

  • Markdown Editor: Fixed bug that prevented VuePress users from writing custom containers.
  • Menus: Fixed bug that incremented menu-weights on re-saving new menu items Added horizontal scroll to menus if they overflowed off-screen
