Today, we announce the first beta of the open-sourced version of Yakkl which can be found at [Yakkl Github Repo](https://github.com/yakkl/yakkl). This beta release will give you a glimpse into what is coming and it includes not only channels but topics so you do not have to put up with "Channel Sprawl".

### Geek Speak

The coming beta release will also remove some of the older Puppet deployment options and bash scripts. The new release will be a docker container of the application and services that will run in the Epiphany platform.

Coming in September will be the full cloud hosted version so all you have to do is create an account for free and use the web version or download the app to your desktop/laptop or mobile device.

Keep Yakkling!
